📊 Sorting and Filtering a Report in Power BI
📌 Introduction

This project demonstrates how to apply sorting and filtering techniques in Microsoft Power BI to improve report usability and 
help stakeholders make better data-driven decisions.

The report is based on a marketing dataset from Adventure Works and focuses on refining visuals using filtering and sorting features.

🎯 Objectives
Understand report navigation in Power BI
Apply sorting to different visuals
Use filters to refine data
Improve clarity and decision-making through clean visuals
🛠️ Tools Used
Microsoft Power BI Desktop
📂 Dataset
Marketing dataset (Adventure Works)
Includes fields like:
Month
Region
Social Media
Order Status
Number of Leads
⚙️ Steps Performed
1. Open Report
Opened the provided .pbix file in Power BI Desktop
2. Sort Ribbon Chart
Sorted by Month → Ascending (Chronological Order)
3. Filter Ribbon & Table Charts
Applied filter: Order Status = Shipped
4. Sort Line Chart
Sorted by Sum of Number of Leads → Ascending
5. Sort Visuals by Region
Stacked Column Chart → Sorted by Region (Ascending)
Table → Sorted by Product Region (Ascending)
6. Filter Social Media (Charts)
Stacked Column Chart & Line Chart filtered for:
Facebook
Google
7. Filter Table Data
Table filtered for:
Facebook
Google
8. Save Report
Saved the updated report for future use
📸 Final Output

The final report includes:

Ribbon Chart (Chronological + Shipped Orders)
Line Chart (Sorted by Leads)
Stacked Column Chart (Filtered Social Media)
Table (Filtered & Sorted Data)
💡 Key Learnings
Sorting improves data readability
Filtering helps focus on relevant insights
Clean visuals enhance decision-making
Power BI enables interactive data exploration
🚀 Conclusion

This project highlights the importance of sorting and filtering in Power BI to transform raw data into meaningful insights. 
These techniques are essential for creating effective and user-friendly reports.
